## Manually
*Monocraft: https://github.com/IdreesInc/Monocraft
*OpenTabletDriver: https://github.com/OpenTabletDriver/OpenTabletDriver
  *mkdir -p ./otd | tar zxvf <directory_of_tarball_file> -C ./otd
rm <directory_of_tarball_file>
cp ./otd/opentabletdriver/etc/udev/rules.d /etc/udev
rm /usr/local/bin
rm /usr/local/lib
rm /usr/local/share
cp ./otd/opentabletdriver/usr/local/bin /usr/local
cp ./otd/opentabletdriver/usr/local/lib /usr/local
cp ./otd/opentabletdriver/usr/local/share /usr/local
rm ./otd

## Summon Ranger Config Command
ranger --copy-config=all

## Quick Monitor Change
xrandr --output <primary_output> --mode 1920x1080 --output <secondary_output> --mode 1920x1080 --right-of <primary_output>

## A Way to fix Flatpak Apps
add "export $(dbus-launch)" to /etc/profile

## Flatpak Commands
*flatpak override --filesystem=home com.example.App
*flatpak run --socket=pulseaudio com.example.App

## Text Editor Default
